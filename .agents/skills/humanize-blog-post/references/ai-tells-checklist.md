# AI-Tells Quick Audit Checklist

Use this checklist to verify a post has been properly humanized. Every box must be checked before the post ships.

## Phrase Removal
- [ ] Zero instances of: Moreover, Furthermore, Additionally, In addition
- [ ] Zero instances of: It is worth noting, In conclusion, To summarize
- [ ] Zero instances of: In the world of, In today's landscape, As we've discussed
- [ ] Zero instances of: delve, tapestry, landscape (metaphorical), leverage (as verb)
- [ ] Zero instances of: seamlessly, cutting-edge, next-generation, holistic
- [ ] Zero instances of: paradigm, ecosystem (non-technical), synergy/synergize
- [ ] Zero instances of: robust/comprehensive/crucial without specific numbers backing them

## Hedging Removal
- [ ] No "may potentially", "could possibly", "might prove"
- [ ] No "it seems that", "it appears that", "it could be argued"
- [ ] No "under certain conditions", "in some cases" (unless genuinely conditional)
- [ ] Genuine uncertainty stated directly: "I'm not sure" / "my guess is"

## Structural Variety
- [ ] No 5+ consecutive sentences of similar word count (14-18 words)
- [ ] At least 2 single-sentence paragraphs for emphasis
- [ ] No 3+ consecutive paragraphs of similar length
- [ ] Perfect parallelism broken (no triple-clause patterns back-to-back)
- [ ] Section headers vary in style (not all "The X of Y" pattern)

## Voice
- [ ] Contractions used throughout (I'm, it's, don't, can't, we've, that's)
- [ ] First person present ("I built", "I found", "I broke")
- [ ] At least 3 questions to reader
- [ ] At least 1 self-correction or admitted uncertainty
- [ ] At least 1 conversational aside or personal reaction
- [ ] At least 1 strong opinion stated without hedging
- [ ] Active voice dominant (not "was implemented", but "I built")

## Opening
- [ ] NOT "In this article/post we will..."
- [ ] NOT "Let's delve/explore/examine..."
- [ ] NOT "The purpose of this post is..."
- [ ] NOT any variant of "In today's world/landscape..."
- [ ] IS one of: specific moment, result first, question, contradiction, data point

## Closing
- [ ] NOT "In conclusion..."
- [ ] NOT "To summarize..."
- [ ] NOT "As we've discussed/seen..."
- [ ] IS one of: circle back, what's next, honest admission, direct recommendation

## Code & Technical Content
- [ ] Code blocks preserved exactly (no modifications to code)
- [ ] Mermaid diagrams preserved exactly
- [ ] Frontmatter YAML preserved exactly
- [ ] Metrics and numbers preserved exactly
- [ ] At least some code shown BEFORE its explanation
- [ ] AI-sounding code introductions replaced ("The following demonstrates..." → "Here's the gate check:")

## Final Read-Aloud Test
- [ ] Read 3 random paragraphs aloud — would you say this to a colleague?
- [ ] No sentence makes you cringe with its formality
- [ ] The post has personality — you can tell a human wrote it
